# Copywriting

## Overview

Use this skill for both new marketing copy and revisions to existing copy.
Default to delivering usable copy first and commentary second.

## When to Use

- Writing new marketing copy from scratch
- Revising existing marketing copy
- Improving headlines, subheads, CTAs, section copy, or page copy
- Clarifying positioning or messaging on product surfaces

## When Not to Use

- Use `seo-audit` when the main issue is ranking or crawlability
- Use `frontend-design` when the main task is layout or visual design

## Minimal Workflow

1. Understand the audience, offer, and desired action.
2. Clarify the promise, proof, and objections.
3. Draft or revise the copy with a clear hierarchy.
4. Tighten for clarity and specificity.

## Reference Routing

- Read [EDITING.md](EDITING.md) when revising existing copy rather than writing from scratch.

## Failure Modes

- Treating every copy task like a from-scratch page rewrite
- Returning annotations and meta-explanations when the user mainly needs usable copy
