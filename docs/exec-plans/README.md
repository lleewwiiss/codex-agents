# Exec Plans

Use this folder for local, durable execution plans when chat alone is not enough.

- `active/`: current plans
- `completed/`: finished plans kept for reference
- `TEMPLATE.md`: minimal plan shape

Use a plan file for:

- large work
- multi-session work
- multi-agent work
- repo-wide reviews or migrations

Keep plans terse.
Keep them local by default.
Commit only when the plan has durable team value or is part of the deliverable.
